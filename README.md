## ğŸ“ Project Overview

The **Virtual Classroom App** is a **front-end focused web application** that allows **teachers and students** to interact in a digital classroom environment.  
It supports:

- Role-based access: **Teacher** vs **Student**
- Assignment creation, submission, and grading
- Uploading and downloading materials (PDF, DOC, PPT)
- Quizzes (single-question format)
- Announcements and discussion forum
- Persistent storage using **localStorage** (no backend required)

This project is **responsive**, visually appealing, and built entirely with **HTML, CSS, and JavaScript**.
